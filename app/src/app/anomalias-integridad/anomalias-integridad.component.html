Descripción

La tabla muestra si existe alguna acción que se realice cuando se ejecuta un UPDATE o DELETE

  • SET NULL: Los datos secundarios se establecen en NULL cuando se eliminan o actualizan los datos princiapales.
  • CASCADE: Los datos secundarios se eliminan o actualizan cuando se eliminan o actualizan los datos principales
  • SET DEFAULT: Los datos secundarios se establecen en sus valores por defecto cuando se eliminan o actualizan los datos principales
  • NO ACTION: No se realiza ninguna acción con los datos secundarios cuando se eliminan o se actualizan los datos principales
